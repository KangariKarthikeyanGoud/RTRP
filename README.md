
# 🎓 Student Performance Analyzer

## 📖 About

Student Performance Analyzer is a web application that helps colleges and schools analyze student academic performance. It converts student data into easy-to-understand charts, reports, and dashboards so that faculty and management can make better decisions.

---

## ✨ Features

- Secure Login
- Student Dashboard
- Performance Analysis
- Attendance Analysis
- Charts & Graphs
- Search & Filters
- Upload CSV Files
- Download Reports
- Responsive Design
